# Forest Fire Prediction

## Project Overview
This project predicts the Fire Weather Index (FWI) using Machine Learning. The model is deployed using Flask and AWS Elastic Beanstalk.

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- AWS Elastic Beanstalk
- GitHub

## Features
- Predicts Fire Weather Index (FWI)
- Web interface using Flask
- Deployed on AWS Cloud

## Project Structure
```
models/
templates/
notebooks/
application.py
requirements.txt
```

## Deployment
The application is deployed on AWS Elastic Beanstalk.
